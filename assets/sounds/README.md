# 音效文件说明

请将点击音效文件放在此目录下，命名为 `bead-click.mp3`

## 推荐音效类型

- 木鱼敲击声
- 念珠碰撞声
- 清脆的敲击声

## 音效要求

- 格式：MP3
- 时长：0.5秒以内
- 大小：建议 50KB 以内

## 免费音效资源

以下网站提供免费音效下载（请遵守各网站的使用条款）：

- [Freesound](https://freesound.org/) - 搜索 "wooden fish" 或 "wood knock"
- [Pixabay](https://pixabay.com/sound-effects/) - 搜索 "click" 或 "tap"
- [Mixkit](https://mixkit.co/free-sound-effects/) - 免费音效库

## 如何替换

1. 准备好你的音效文件
2. 将文件重命名为 `bead-click.mp3`
3. 将文件放入 `assets/sounds/` 目录
4. 重新编译小程序即可生效

## 音效处理建议

如果音效文件过大，可以使用以下工具压缩：

- [Online Audio Converter](https://online-audio-converter.com/)
- [Audacity](https://www.audacityteam.org/) - 开源音频编辑软件

建议操作：

1. 裁剪至 0.3-0.5 秒
2. 降低比特率至 64kbps
3. 转换为单声道
