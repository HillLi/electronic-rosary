# 电子念珠微信小程序 - 需求文档

> 创建日期：2026-03-09
> 状态：需求确认完成，待开发

---

## 一、项目概述

### 1.1 项目背景
开发一款电子念珠微信小程序，模拟真实佛珠/念珠的使用体验，让用户可以通过手机进行念诵计数。

### 1.2 核心理念
- 模拟真实念珠的触感和视觉体验
- 操作简单，点击即可计数
- 支持个性化设置（滚动方向）

---

## 二、功能需求

### 2.1 主界面

| 功能 | 描述 |
|------|------|
| 念珠展示 | 以串珠形式垂直排列展示，屏幕显示7颗珠子，中间高亮 |
| 功德计数器 | 顶部显示累计功德数字，+1时有弹跳动画效果 |
| 点击交互 | 点击屏幕任意位置，整串念珠滚动一颗，功德+1 |
| 设置入口 | 页面底部显示设置按钮 |
| 珠子样式 | 圆形珠子，无连接线，简洁风格 |

### 2.2 念珠滚动动画

| 属性 | 说明 |
|------|------|
| 动画类型 | 整串念珠同时滚动，非单颗切换 |
| 滚动方向 | 支持两种方向，用户可在设置中切换 |
| 循环方式 | 环形循环，无固定数量限制，模拟真实手串 |
| 动画时长 | 约 300ms，流畅自然 |

**方向说明：**
- **从下往上 ↑**：点击后珠子向上滚动，如同把珠子推上去
- **从上往下 ↓**：点击后珠子向下滚动，如同重力牵引

### 2.3 音效

| 属性 | 说明 |
|------|------|
| 触发时机 | 每次点击时播放 |
| 音效类型 | 木鱼声或念珠碰撞声（用户自行替换） |
| 技术方案 | 使用 `wx.createInnerAudioContext()` |
| 文件格式 | MP3 |

### 2.4 设置页面

| 设置项 | 选项 |
|--------|------|
| 滚动方向 | ○ 从下往上 ↑（推动感）<br>○ 从上往下 ↓（重力感） |
| 音效开关 | ○ 开启<br>○ 关闭 |

## 三、非功能需求

### 3.1 视觉风格

**整体风格：** 真实木质风格

**配色方案：**
| 元素 | 颜色 | 说明 |
|------|------|------|
| 背景 | #F5E6D3 | 米黄色，仿佛珠袋 |
| 念珠 | #8B4513 | 深棕色，带渐变光泽 |
| 当前珠子高亮 | #FFD700 | 金色光环 |
| 文字 | #5D4037 | 深褐色 |

**念珠样式：**
- 圆形珠子
- 径向渐变模拟立体感
- 轻微阴影增加层次
- 当前珠子有脉冲光晕动画

### 3.2 数据存储

| 数据 | 存储方式 | 说明 |
|------|----------|------|
| 总功德数 | wx.setStorageSync | 永久保存本地 |
| 设置项 | wx.setStorageSync | 方向等设置 |

### 3.3 性能要求
- 动画流畅，无卡顿
- 循环衔接自然（首尾珠子无缝循环）
- 防止动画期间重复点击

## 六、技术方案概要

### 6.1 项目结构
```
electronic-rosary/
├── app.js                    # 全局逻辑
├── app.json                  # 小程序配置
├── app.wxss                  # 全局样式
├── sitemap.json              # 索引配置
├── assets/
│   └── sounds/
│       └── bead-click.mp3    # 点击音效
└── pages/
    ├── index/                # 主页
    │   ├── index.js
    │   ├── index.json
    │   ├── index.wxml
    │   └── index.wxss
    └── settings/             # 设置页
        ├── settings.js
        ├── settings.json
        ├── settings.wxml
        └── settings.wxss
```

### 6.2 核心数据结构
```javascript
data: {
  merit: 0,                    // 总功德
  beads: [0,1,2,3,4,5,6],      // 显示的7颗珠子索引
  currentIndex: 3,             // 当前高亮珠子（中间位置）
  direction: 'up',             // 滚动方向: 'up' | 'down'
  soundEnabled: true,          // 音效开关
  isAnimating: false           // 动画状态锁
}
```
